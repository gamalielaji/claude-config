# Eval Task: Creative Director Agent

## Scenario
The game-designer and art-director disagree on art style. Game designer wants
pixel art for nostalgia and scope efficiency. Art director wants hand-painted
2D for visual distinction. The game is a narrative-driven adventure.

## Expected Behavior
The agent should:
1. Acknowledge both perspectives with respect
2. Evaluate each option against game pillars / vision
3. Consider practical constraints (team size, timeline, budget)
4. Make a BINDING decision with clear reasoning
5. Frame the decision to maintain team morale
6. NOT punt ("you two should figure it out")

## Scoring Rubric
| Criterion | Points | Pass Condition |
|-----------|--------|----------------|
| Acknowledges both sides | 2 | Summarizes each argument fairly |
| Evaluates against vision | 3 | References game pillars or core identity |
| Considers constraints | 2 | Mentions scope, team, timeline |
| Makes binding decision | 3 | Clear "we will do X" statement |
| Preserves team morale | 2 | Validates losing side, explains why |
| Authoritative tone | 1 | Speaks as final creative authority |
| **Total** | **13** | |

## Anti-Patterns (instant fail)
- Refuses to decide ("both are valid, you choose")
- Ignores one side's argument entirely
- Makes decision without reasoning
